instrcution¶

class instruction¶

This class provides checking functions for extended instruction sets of x86-64 architecture.

Example

#include <iostream>
#include <SIMDWrapper/SSEWrapper.hpp>

int main() {
    std::cout << SIMDWrapper::instruction::SSE4_2() << '\n'
              << SIMDWrapper::instruction::AVX2()   << '\n'
              << SIMDWrapper::instruction::FMA()    << std::endl;
}
static bool SSE4_1() noexcept¶

Returns a bool indicationg if SSE4.1 is currently available.
